This is a [Next.js](https://nextjs.org) project for **Hostopia**: the main marketing site and the **Marketing Dashboard**.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) for the main site and [http://localhost:3000/marketing](http://localhost:3000/marketing) for the Marketing Dashboard.

## Project structure

- **`/`** — Hostopia marketing site (platform, solutions, contact).
- **`/marketing`** — Marketing Dashboard: website traffic, metrics, and trend charts. Switch between brands via the header dropdown.
